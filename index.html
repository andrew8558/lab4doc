Лабораторная работа 4

Login

Путь: /login

Страница для входа в систему. На странице находится форма с паролем и логином и кнопка для отправки формы на эндпоинт для получения токена.

Registration

Путь: /registration

Страница с регистрацией. На странице находится форма с полями: логин, почта, пароль, повтор пароля, и кнопка для отправки формы на эндпоинт создания пользователя.

Home

Путь: /home

Домашняя страница со списком из доступных разделов.

Teachers

Путь: /teachers

Страница со списком учителей, выбрав конкретного учителя, можно перейти на его страницу.

Teacher

Путь: /teachers/:id

Страница с информацией об учителе и его расписании, есть кнопка для создания формы для редактирования данных об учителе, кнопка для отправки формы на эндпоинт для изменения данных, и кнопка удаления объекта учителя, отправляющая запрос на эндпоинт для удаления учителя.

Classes

Путь: /classes

Страница с таблицей классов и их классных руководителей, выбрав конкретный класс, можно перейти на его страницу.

Class

Путь: /classes/:id

Страница с информацией о класса, списке учеников, и отчете об успеваимости в виде таблице со столбцами: предмет, средняя оценка по предмету. Выбрав конкретного ученика, можно перейти на его страницу.

Student

Путь: /students/:id

Страница с информацией о ученике в виде редактируемой формы, его оценками по каждому предмету за четверть. Есть кнопка для отправки формы на эндпоинт для изменения данных, и кнопка удаления объекта ученика, отправляющая запрос на эндпоинт для удаления ученика. Также можно изменять оценки ученика по конкретному предмету, при измении отправляетя запрос на эндпоинт изменения оценки.

Add teacher

Путь: /add-teacher

Страница с формой для добавления нового учителя, отправляющей запрпос на эндпоинт для внесения данных о новом учителе.

Add student

Путь: /add-student

Страница с формой для добавления нового ученика, отправляющей запрпос на эндпоинт для внесения данных о новом ученике.

Add lesson

Путь: /add-lesson

Страница с формой для создания урока в расписании, отправляющей запрос на эндпоинт для внесения нового урока в расписание.
